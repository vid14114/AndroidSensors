AndroidSensors
==============
A program for reading all sensor values and display them in many ways

1) Sensors
	* Movement
	* Accelerometer
	* Magnet
	* Direction
	* Air Pressure - online weather
	* Temperature - online weather
	* Light
	* (Microphone)
	* (Kamera)
	* (GPS)

2) Output
	* XML
	* Views
	* Dialogs
	* Toasts
	* Audiofiles
	* TTS
	* Vibration
	* E-Mail
	* Phone call
	* SMS
	* (File)
	* (Internet)

3) Input elements
	* Keyboard
	* Touch
	* Movement
	* (Language)
	* (Kamera)

()... nice to have
